# Excel Data Cleaning - Netflix Dataset

## Objective
To clean and prepare a real-world dataset using Excel by handling missing values, duplicates, and inconsistent formats.

## Dataset
Netflix Movies and TV Shows dataset from Kaggle.

## Steps Performed
1. Identified and highlighted missing values (Director, Cast, Country).
2. Removed duplicate records after creating a backup.
3. Cleaned text using TRIM and PROPER functions.
4. Standardized country names and formats.
5. Created a separate Cleaned_Data sheet.
6. Added Data Quality Notes for analysis transparency.

## Tools Used
- Excel (Web)
- LibreOffice Calc (optional backup)
- GitHub

## Deliverables
- Raw_Data.xlsx  
- Cleaned_dataset.xlsx  
- cleaned_dataset.csv  

## Learning Outcome
Understood real-world data cleaning workflow followed by data analysts before analysis and visualization.
